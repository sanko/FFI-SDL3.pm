=head1 NAME

SDL2::Version - Information About the Version of SDL in Use

=head1 SYNOPSIS

    use SDL2::FFI qw[:version];
    my $ver = SDL_GetVersion();
    CORE::say sprintf 'SDL version %d.%d.%d', $ver->major, $ver->minor, $ver->patchlevel;

=head1 DESCRIPTION

SDL2::Version represents the library's version as three levels: major, minor,
and patchlevel.

=head1 Methods

=head2 C<major( )>

Returns value which increments with massive changes, additions, and
enhancements.

=head2 C<minor( )>

Returns value which increments with backwards-compatible changes to the major
revision.

=head2 C<patch( )>

Returns value which increments with fixes to the minor revisionl.
