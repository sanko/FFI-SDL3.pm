=head1 NAME

SDL3::Image::Animation - SDL Animated Image Support

=head1 SYNOPSIS

    use SDL3 qw[:assert];

=head1 DESCRIPTION

Currently only animated GIFs are supported.

=head1 Fields

=over

=item C<w> - image width

=item C<h> - image height

=item C<count> - the number of frames

=item C<frames> - list of L<SDL3::Surface> structures representing individual animation frames

=item C<delays> - list of frame delays

=back
