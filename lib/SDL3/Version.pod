=head1 NAME

SDL3::Version - Information About the Version of SDL in Use

=head1 SYNOPSIS

    use SDL3 qw[:version];
    SDL_GetVersion( my $ver = SDL3::version->new );
    CORE::say sprintf 'SDL version %d.%d.%d', $ver->major, $ver->minor, $ver->patch;

=head1 DESCRIPTION

SDL3::Version represents the library's version as three levels: major, minor,
and patch level.

=head1 Fields

=head2 C<major( )>

Returns value which increments with massive changes, additions, and
enhancements.

=head2 C<minor( )>

Returns value which increments with backwards-compatible changes to the major
revision.

=head2 C<patch( )>

Returns value which increments with fixes to the minor revision.
