=head1 NAME

SDL3::Locale - SDL Locale Structure

=head1 SYNOPSIS

    use SDL3 qw[:locale];
    my $locale = SDL_GetPreferredLocales( );

=head1 DESCRIPTION

SDL3::Locale is a structure containing the user's preferred locale.

=head1 Fields

=over

=item C<language> - a language name; like "C<en>" for English

=item C<country> - a country; like "C<US>" for the United States. Can be C<undef>

=back
