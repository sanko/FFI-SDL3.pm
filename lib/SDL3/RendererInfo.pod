=head1 NAME

SDL3::RendererInfo - Information on the Capabilities of a Render Driver or
Context

=head1 SYNOPSIS

    use SDL3 qw[:renderer];
    # TODO
    warn $renderer->name;

=head1 DESCRIPTION

SDL3::RendererInfo exposes render driver information.

=head1 Fields

=over

=item C<name> - the name of the renderer

=item C<flags> - Supported L<< C<SDL_RendererFlags>|SDL3/C<:rendererFlags> >>

=item C<num_texture_formats> - the number of available texture format

=item C<texture_formats> - the available texture formats

=item C<max_texture_width> - the maximum texture width

=item C<max_texture_height> - the maximum texture height

=back
